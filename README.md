# ramoncunha.dev

My personal portfolio — a dark-themed, single-page business card built with HTML, Tailwind CSS, and vanilla JavaScript.

## ✨ Features

- **Responsive design** — looks great on desktop and mobile
- **Bilingual** — English 🇺🇸 and Portuguese 🇧🇷 with auto-detection based on browser language
- **No frameworks** — pure HTML, CSS, and JS for fast load times
- **Dark aesthetic** — glassmorphism, subtle animations, and grain texture

## 🛠 Tech Stack

- HTML5
- Tailwind CSS (CDN)
- Vanilla JavaScript
- Google Fonts (DM Sans, Playfair Display)
- Font Awesome icons

## 🚀 Running Locally

Just open `index.html` in your browser — no build step required.

## 📁 Structure

```
├── index.html        # Main page
├── assets/           # Profile image
├── js/
│   └── i18n.js       # Language detection & switching
└── locales/
    ├── en.js          # English translations
    └── pt.js          # Portuguese translations
```
