# Luisa — Chá de Bebê

A baby shower gift list built with [Next.js](https://nextjs.org) and React.

## Development

```bash
npm install
npm run dev
```

The app runs at [http://localhost:3000/luisa](http://localhost:3000/luisa).

## Build

```bash
npm run build
```

Static output is generated in the `out/` directory.
